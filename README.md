# 浪漫求婚网站

一个星空梦幻主题的求婚网站，记录罗先生与思宝的爱情故事。

## 功能特色

- 首页：动态文字效果，星空背景
- 时间轴：展示从高中相遇到现在的爱情故事
- 求婚互动：爱情问答、拼图游戏、情书揭秘
- 美好回忆：婚纱照相册

## 启动方式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173/

## 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 添加背景音乐

将音乐文件放到 `public/music/background.mp3`

推荐免版权音乐网站：
- [Pixabay Music](https://pixabay.com/music/)
- [Chosic](https://www.chosic.com/free-music/all/)

搜索关键词：romantic, wedding, piano
